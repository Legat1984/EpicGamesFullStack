# Игровая платформа (Epic Games Clone)

## Описание проекта
Это веб-приложение представляет собой игровую платформу, вдохновлённую интерфейсом Epic Games Store, с возможностью аутентификации пользователей, просмотра библиотеки игр и общения в чате.

## Основные функции
- Регистрация и аутентификация пользователей
- Подтверждение по электронной почте
- Восстановление пароля
- Платформа с библиотекой игр
- Система чатов
- Адаптивный дизайн
- Темная/светлая тема

## Стек технологий
- React 18
- React Router
- Styled Components
- Context API
- JWT токены
- Lucide React (иконки)
- Axios

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Структура проекта
```
src/
├── components/              # Компоненты приложения
│   ├── EpicGames/           # Компоненты для экрана Epic Games (логин и т.д.)
│   └── Platform/            # Компоненты для основной платформы
├── contexts/                # React Context провайдеры
├── style/                   # Глобальные стили и темы
└── assets/                  # Ассеты (изображения, шрифты)
```

## API endpoints
Приложение ожидает backend API на адресе, указанном в `.env` файле.

## Особенности
- Полностью адаптивный дизайн
- Поддержка темной/светлой темы
- Система аутентификации с автоматическим обновлением токенов
- Чат с поддержкой нескольких комнат
- Мобильное меню для адаптивности
